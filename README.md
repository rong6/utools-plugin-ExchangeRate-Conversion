### 这是什么

一个简单的、基于Utools的汇率转换工具，API使用[ExchangeRate-API](https://www.exchangerate-api.com/)。   

需要注意的是，该项目的绝大部分代码均由ChatGPT的GPT-4o模型编写。  

此外，ExchangeRate-API的汇率数据与国内主流平台有一定的误差，请自行判断。
